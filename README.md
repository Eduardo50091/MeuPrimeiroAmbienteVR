# Projeto Final: Meu Primeiro Ambiente VR 🥽

Este projeto é uma experiência inicial em Realidade Virtual desenvolvida para a trilha **Metaverso**. 
O objetivo é demonstrar os fundamentos de XR criando um ambiente navegável e organizado no Unity.

## 🛠️ Configuração Técnica
* **Engine:** Unity (compatível com Meta SDK).
* **Plataforma:** Android (Meta Quest).
* **SDK:** Meta XR SDK e XR Plugin Management configurados.
* **Movimentação:** Funcional via teclado/mouse no PC para facilitar os testes iniciais.

## 🌌 O Ambiente
O cenário foi projetado para simular uma exposição de alguns carros esportivos, contendo:
* **Piso/Terreno:** Base para o usuário ver os carros.
* **Objetos:** Uma garagem para trazer um ambiente familiar, três carros que são atração principal e luzes..
* **Destaque:** Inclusão de modelos detalhados, HUD com informações e sons, criando um ambiente mais realista.

## 📂 Organização do Projeto
Seguindo as boas práticas de desenvolvimento, pastas e objetos para organizar os gameobjects:
* Hierarquia de pastas lógica (Assets, ProjectSettings, Packages).
* Nomenclatura clara de arquivos e objetos.
* Uso de `.gitignore` para manter o repositório leve.

## 🚀 Como Executar
1. Clone o repositório.
2. Certifique-se de ter o **Git LFS** instalado para baixar os modelos pesados.
3. Abra o projeto no Unity.
4. Pressione **Play** para navegar no PC ou faça o **Build & Run** para o Meta Quest.
