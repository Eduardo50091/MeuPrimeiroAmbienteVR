# Projeto Final: Meu Primeiro Ambiente VR 🥽

[cite_start]Este projeto é uma experiência inicial em Realidade Virtual desenvolvida para a trilha **Metaverso**[cite: 2, 3]. 
[cite_start]O objetivo é demonstrar os fundamentos de XR criando um ambiente navegável e organizado no Unity[cite: 6, 7].

## 🛠️ Configuração Técnica
* [cite_start]**Engine:** Unity (compatível com Meta SDK)[cite: 10].
* [cite_start]**Plataforma:** Android (Meta Quest)[cite: 12].
* [cite_start]**SDK:** Meta XR SDK e XR Plugin Management configurados[cite: 11, 13].
* [cite_start]**Movimentação:** Funcional via teclado/mouse no PC para facilitar os testes iniciais[cite: 14].

## 🌌 O Ambiente
[cite_start]O cenário foi projetado para simular uma exposição de alguns carros esportivos[cite: 19], contendo:
* [cite_start]**Piso/Terreno:** Base para o usuário ver os carros[cite: 17].
* [cite_start]**Objetos:** Uma garagem para trazer um ambiente familiar, três carros que são atração principal e luzes.[cite: 16].
* **Destaque:** Inclusão de modelos detalhados, HUD com informações e sons, criando um ambiente mais realista.

## 🚀 Como Executar
1. Clone o repositório.
2. Certifique-se de ter o **Git LFS** instalado para baixar os modelos pesados.
3. Abra o projeto no Unity.
4. Pressione **Play** para navegar no PC ou faça o **Build & Run** para o Meta Quest.
